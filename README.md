# 📱 YBT Store -- Android E-Commerce App

## 📌 Project Description

YBT Store is a simple and user-friendly Android e-commerce application
built using Kotlin. The app allows users to browse products, search
items, add them to a cart, and view their profile.

## 🚀 Features

-   🏠 Home Screen -- Displays products and categories\
-   🔍 Search -- Easily find products using keywords\
-   🛒 Cart -- Add and remove items from cart\
-   👤 Profile -- User information and account section\
-   ⚡ Responsive UI -- Smooth and clean interface

## 🛠️ Tech Stack

-   Language: Kotlin\
-   Architecture: MVVM\
-   UI Design: XML / Jetpack Components\
-   Backend: Firebase\
-   Build Tool: Gradle (Kotlin DSL)

## ⚙️ Setup Instructions

1.  Clone the repository: git clone
    https://github.com/your-username/ybt-store.git

2.  Open in Android Studio\

3.  Sync Gradle\

4.  Add google-services.json\

5.  Run the app
